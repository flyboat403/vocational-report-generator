# 报告章节目录模板

**使用说明**：撰写 Phase 4 开始前完整阅读。按报告类型选择对应模板，章节不可增减。

---

## 一、常规性专业调研报告（7章固定结构）

逻辑链：行业趋势 → 区域产业 → 岗位需求 → 人才供给 → 企业反馈 → 毕业生反馈 → 建设建议

```
# [报告标题]

摘要（可选）

## 一、引言
- 调研背景（行业趋势、政策推动、学校需求）
- 调研目的（专业建设决策需求）
- 调研方法（文献研究、实地走访、问卷访谈等）
- 调研范围（产业/地域/专业/岗位四个维度）

## 二、行业现状分析
- 行业定义与业态特征
- 全国市场规模与增长趋势
- 代表性区域/城市对比分析
- 标杆案例借鉴
- 发展趋势研判

## 三、[区域]基本情况
- 本地资源禀赋与产业政策
- 产业发展现状（核心数据：游客量、收入、增幅等）
- 产业定位与特色优势
- 存在的问题与短板

## 四、专业人才供给分析
- 全国同类专业人才培养现状
- 本地/本校人才供给现状（招生、就业）
- 各专业当前培养方向与行业需求的匹配度
- 人才供给的共性问题

## 五、用人单位需求调研
- 调研企业基本情况
- 核心能力需求分析（排名+差距分析）
- 各岗位群典型工作任务与职业能力
- 企业招聘困难与期望
- **数据来源必须标注**：【校方调研数据】（用户提供，措辞用"本次调研"）/【公开数据推断】（agent 推断，措辞用"据XX平台数据"）/【留空】（标注"建议校方补充调研"）

## 六、毕业生就业情况调研
- 毕业生调研概况
- 就业数据（就业率、对口率、薪资、离职率）
- 毕业生对学校教育的反馈
- 对专业建设的建议
- **数据来源必须标注**：【校方就业质量报告】（用户提供）/【公开就业报告推断】（agent 推断，措辞用"据XX就业报告"）/【留空】（标注"建议校方补充调研"）。禁止编造具体百分比却无来源标注。

## 七、专业群/专业建设建议
- 核心结论（6条以内）
- 课程体系建议
- 实训条件建议
- 师资队伍建设建议
- 校企合作建议
- 分阶段建设路线
- 预期成效指标
```

## 二、双高专业群可行性分析报告（10章固定结构）

逻辑链：国家战略 → 区域需求 → 产业微观 → 大数据分析 → 专业关联 → 组群逻辑 → 优势对比 → 可行性论证 → 建设建议

完整章节目录见 `references/majors_report_structor.md`。该文件包含每章必须覆盖的内容要点和详细子节。

```
# [报告标题]

## 一、专业群建设基础情况
- 学校基本情况（办学定位、历史沿革、优势特色）
- 相关专业建设现状（已有专业、师资、实训条件）
- 专业群建设的现实动因（产业需求、政策驱动、学校发展）

## 二、国家战略需求分析
- 国家战略方向对接分析（13个强国战略、5个中国战略）
- 区域战略协调性分析（12个重大战略区域、19个国家级新区）
- 社会服务面向定位

## 三、区域需求发展分析
- 区域资源禀赋与产业基础
- 区域"十五五"产业规划与政策导向
- 区域对相关人才的需求紧迫性
- 专业群建设与区域发展的匹配度

## 四、产业微观分析
- 产业链分析：核心产业链定位（14条国家核心产业链）
- 技术链分析：关键技术环节与技能要求
- 人才链分析：人才供给与需求的缺口判断
- 产业链→技术链→人才链的映射关系

## 五、产业人才大数据分析
- 人才需求现状分析（地域分布、学历要求、经验要求、薪资水平）
- 人才需求趋势预测（近3年岗位发布量变化趋势）
- 人才供给分析（同类专业毕业生输出量、区域人才存量）
- 岗位任务能力分析（典型岗位的工作任务、知识/技能/工具要求）

## 六、产业关联专业分析
- **产业关联专业（教育部目录层面，可推断）**：与产业强关联的专业目录匹配（中职→高职→本科贯通）
- **本校已开设专业（学校事实，须用户提供）**：现有专业的覆盖度分析
- 专业关联岗位模型匹配（知识、要求、方法、技能、工具五维度）
- **两层必须分两张表呈现，禁止合并**——产业关联专业表标注"教育部目录层面"，本校已开设专业表标注"校方提供/agent 推断"

## 七、专业群组群逻辑分析
- 组群逻辑主线（产业逻辑/岗位逻辑）
- 核心专业与支撑专业的层次关系
  - **如走用户提供路径**：基于校方已开设专业做组群论证（事实描述，措辞用"本校已开设"）
  - **如走 agent 推断路径**：基于产业链/岗位链推断支撑专业，措辞用"建议纳入"，首处标注"（推断方案，待校方确认）"
- 各专业间的课程共享、师资共享、实训共享可行性
- 产教对接谱系分析

## 八、申报优势对比分析
- 横向对比：同类院校同类专业群的建设水平对比
- 纵向对比：本校相关专业群的成长与变化趋势
- 差异化优势提炼

## 九、专业群设置的可行性分析
- 产业需求可行性、政策可行性、生源可行性
- 师资可行性、实训条件可行性、就业前景可行性

## 十、调研结论与建设建议
- 核心结论（"建什么、怎么建"的判断）
- 专业群建设实施路径
- 预期成效与标志性成果
- 政策保障与资源需求
```
